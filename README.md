# CMA AI Mentor

An AI-powered CMA (Cost & Management Accounting) learning platform built using **Streamlit** and **Large Language Models**.

## Live:
https://cmamentor-rx4wmay6vtqy6jdmvkeu6h.streamlit.app/CMA_Chatbot

## Features

- **AI Chatbot (CMA Mentor)**
  - Answers CMA syllabus-based questions
  - Exam-oriented explanations
  - Uses free LLM (Groq – LLaMA 3.1)

- **AI Mock Tests**
  - Topic-based CMA mock tests
  - Difficulty selection (Easy / Medium / Hard)
  - Custom instructions (numericals, PYQ style, tricky questions)
  - Auto-evaluation with explanations

- **AI Previous Year Papers**
  - AI-generated PYQ-style question papers
  - CMA exam pattern (Section A / B / C)
  - User-controlled difficulty and style

## Tech Stack

- Python
- Streamlit
- Groq API (LLaMA 3.1 – free)
- Prompt Engineering

##  Note

This project uses **AI-generated PYQ-style questions**.  
It does **not distribute official copyrighted CMA papers**.
